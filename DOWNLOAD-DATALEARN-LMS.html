‚ö° Quick Start

  1. Download & Extract: Download file ZIP di atas, extract ke folder project Anda
  2. Install Software: Install Node.js 16+ dan PostgreSQL 14+
  3. Setup Database:
    psql -U postgres
    CREATE DATABASE datalearn_lms;
    psql -U postgres -d datalearn_lms -f database/schema.sql
  4. Setup Backend:
    cd backend && npm install
    cp .env.example .env
    (Edit .env, isi DB_PASSWORD dan JWT_SECRET)
    npm run dev
  5. Buka Frontend: Buka file frontend/index.html di browser
  6. Login: Email: admin@datalearn.com Password: admin123
